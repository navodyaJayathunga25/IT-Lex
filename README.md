# IT-Lex

## Description
IT Lex – A web-based IT word game that helps players learn English and IT terms through interactive games, hints, and bookmark services

## Tech Stack
- MongoDB
- Express
- React
- Node.js

## Installation

Clone repo

git clone https://github.com/navodyaJayathunga25/IT-Lex.git

Install dependencies

cd backend
npm install

cd ../frontend
npm install

## Run Project

Backend
npm run dev

Frontend
npm run dev
